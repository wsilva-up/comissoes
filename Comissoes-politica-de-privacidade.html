ComissõesPro

Política de Privacidade

Última atualização: junho de 2026

Esta Política de Privacidade descreve como o aplicativo ComissõesPro, desenvolvido por Walter A. da Silva, coleta, usa e protege as informações dos usuários.

O ComissõesPro é um aplicativo de uso profissional restrito, destinado exclusivamente a usuários autorizados pelo desenvolvedor. O acesso é controlado individualmente por e-mail.

1. Informações coletadas

O ComissõesPro coleta apenas as informações estritamente necessárias para o funcionamento do app:

  • Nome e e-mail Google: obtidos no momento do login via Google OAuth, usados exclusivamente para verificar se o usuário tem acesso autorizado.
  • Foto de perfil Google: exibida no cabeçalho do app apenas para identificação visual do usuário logado.
  • Dados de vendas: informações inseridas manualmente pelo usuário (valores, clientes, veículos, comissões), armazenadas localmente no dispositivo.

2. Como as informações são usadas

  • O e-mail é verificado contra uma lista de usuários autorizados mantida pelo desenvolvedor.
  • Os dados de vendas são armazenados localmente no navegador do dispositivo do usuário (localStorage) e não são enviados a servidores externos.
  • Nenhuma informação é compartilhada com terceiros, utilizada para fins publicitários ou vendida a qualquer empresa.

3. Armazenamento de dados

Todos os dados de vendas ficam armazenados exclusivamente no dispositivo do usuário. O desenvolvedor não tem acesso ao histórico de vendas, configurações ou qualquer dado inserido no app.

O usuário pode exportar, importar ou apagar seus dados a qualquer momento diretamente pelo app, na aba Dados.

4. Autenticação Google

O login é realizado via Google OAuth 2.0. O ComissõesPro solicita apenas permissões básicas de perfil (nome, e-mail e foto). Nenhuma permissão de acesso a Gmail, Google Drive, contatos ou qualquer outro serviço Google é solicitada ou utilizada.

O processo de autenticação é gerenciado integralmente pelo Google e está sujeito à Política de Privacidade do Google.

5. Controle de acesso

O acesso ao app é concedido individualmente pelo desenvolvedor. Usuários não autorizados não conseguem acessar o aplicativo mesmo que possuam o link. O desenvolvedor pode revogar o acesso de qualquer usuário a qualquer momento.

6. Segurança

O ComissõesPro adota as seguintes medidas de segurança:

  • Autenticação via Google OAuth — não armazenamos senhas
  • Acesso restrito a usuários previamente autorizados
  • Dados armazenados localmente no dispositivo do usuário
  • Comunicação via HTTPS em todas as requisições

7. Direitos do usuário

O usuário tem direito a:

  • Exportar todos os seus dados a qualquer momento (aba Dados → Exportar)
  • Apagar todos os seus dados do dispositivo (aba Dados → Limpar histórico)
  • Solicitar a remoção do seu acesso entrando em contato com o desenvolvedor

8. Cookies e rastreamento

O ComissõesPro não utiliza cookies de rastreamento, ferramentas de análise de comportamento ou qualquer tecnologia de monitoramento de usuários.

9. Alterações nesta política

Esta política pode ser atualizada periodicamente. Alterações significativas serão comunicadas aos usuários autorizados. A data da última atualização está sempre indicada no topo desta página.
